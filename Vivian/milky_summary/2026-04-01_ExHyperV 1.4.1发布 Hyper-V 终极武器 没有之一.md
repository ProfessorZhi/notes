# ExHyperV 1.4.1 发布笔记

## 项目概述

**ExHyperV** 是一款专为 Windows Hyper-V 打造的图形化管理工具，基于 **WPF-UI** 框架开发，采用现代化的科幻风格界面设计。

- **项目性质**：完全开源免费
- **开发框架**：WPF-UI
- **GitHub 地址**：https://github.com/Justsenger/ExHyperV
- **最新版本下载**：https://github.com/Justsenger/ExHyperV/releases/latest

---

## 核心功能特性

### 1. 现代化 UI 界面

- 流畅的科幻视觉效果
- 支持 **深色/浅色主题自动切换**

### 2. GPU-PV 管理（显卡半虚拟化）

- 图形化配置显卡分区
- 支持系统：Windows 10/11 及 Windows Server
- 配合 **Parsec** 或 **Moonlight** 可实现虚拟机高性能云游戏

> **弹幕反馈**：用户对该功能反响强烈，有弹幕称解决了"Linux 与 NVIDIA 的历史冲突"

### 3. 高级硬件设置

| 功能 | 说明 |
|------|------|
| CPU 拓扑配置 | 可视化调整虚拟机 CPU 核心/线程分配 |
| 嵌套虚拟化 | 一键启用嵌套虚拟化支持 |
| PCIe 设备直通 (DDA) | 图形化配置 PCIe 设备直通 |

### 4. 存储与网络管理

- **磁盘读写监控**：直观的实时 I/O 监控界面
- **VLAN 隔离**：图形化配置虚拟网络隔离
- **NAT 网络**：一键配置 NAT 网络

### 5. Linux 支持

- 帮助 Ubuntu、Arch 等 Linux 虚拟机安装驱动

## 安装方式

### 方式一：Winget 安装（推荐）

```powershell
winget install ExHyperv
```

### 方式二：GitHub 下载安装

1. 访问 https://github.com/Justsenger/ExHyperV/releases/latest
2. 下载最新版本的安装包
3. 运行安装程序

---

## 适用场景

1. **游戏云化**：通过 GPU-PV 将显卡资源分配给虚拟机，配合 Parsec/Moonlight 实现远程游戏
2. **开发测试**：快速配置嵌套虚拟化环境进行 Hyper-V/Android 模拟器开发
3. **硬件直通**：将物理 GPU 或其他 PCIe 设备直通给虚拟机使用
4. **网络实验**：快速搭建 VLAN 隔离或 NAT 网络环境
5. **Linux 虚拟机管理**：简化 Linux 虚拟机的驱动安装和配置
