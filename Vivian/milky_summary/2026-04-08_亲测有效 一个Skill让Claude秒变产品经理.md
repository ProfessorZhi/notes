# Product Manager Skills：让Claude秒变资深产品经理

## 项目概述

**Product Manager Skills** 是一款开源在 GitHub 上的 Claude Skill，由一位资深产品经理贡献。该项目将真实的产品经理工作方法论拆解成 46 个 Claude 能够理解的 Skill 模块，装上后 Claude 会像「读过这些方法论的资深 PM」一样，主动按框架引导用户工作。

> **核心定位**：本质是一套**决策框架**，适用于任何需要需求分析、优先级排序的场景。

## 项目特点

| 特点 | 说明 |
|------|------|
| 框架数量 | 46 个产品管理框架 |
| 专业深度 | 基于真实 PM 工作流，非通用建议 |
| 使用方式 | 安装后 Claude 自动识别场景并调用对应框架 |
| 开源属性 | GitHub 开源项目 |

## 核心功能示例：RICE 优先级排序

视频演示了 RICE 框架的使用效果：

**输入**：5 个产品需求

**输出**：Claude 按照 RICE 的 4 个维度对每个需求逐个打分：

| 维度 | 英文 | 含义 |
|------|------|------|
| Reach | 触达 | 该需求能影响多少用户 |
| Impact | 影响 | 对用户/业务的影响程度 |
| Confidence | 信心 | 对估算的确定程度（百分比） |
| Effort | 努力 | 团队需要投入的人月数 |

最终给出：
- 综合排序结果
- **Quick Win** 标注（投入少、影响大的快速胜利项）
- **BFI**（Big Fat Ideas）标注（高投入高回报的核心项目）

## 支持的框架类型（部分）

- RICE 优先级排序
- 用户洞察分析
- 用户故事（User Story）撰写
- 竞品分析
- 需求池管理
- PRD 文档结构化

## 适用场景

| 角色 | 使用场景 |
|------|----------|
| 产品经理 | PRD 撰写、需求分析、优先级决策 |
| 运营人员 | 用户洞察分析、用户分层 |
| 创业者 | 产品定位梳理、商业模式验证 |
| 项目经理 | 需求池管理、迭代规划 |

## 安装与使用

1. 在 Claude 中加载 Product Manager Skills 项目
2. 描述你的需求或任务（如「我有 5 个需求，请用 RICE 框架排序」）
3. Claude 自动调用对应的 PM 框架进行专业分析

## 总结

如果你有**需求分析、优先级排序、用户洞察**等场景需求，这套 Skill 包能显著提升 AI 输出的专业度和实用性。46 个框架覆盖了产品管理工作的各个环节，让 AI 不再只是「写通用文案的工具」，而是真正懂产品方法论的搭档。
