# Code Panorama - AI 源码全景图生成工具

## 项目信息

| 项目 | 内容 |
|------|------|
| 工具名称 | Code Panorama（代码全景图） |
| GitHub 地址 | https://github.com/xuanyuanzhifeng/code-panorama |
| 功能定位 | 代码分析智能体 |

## 核心功能

### 输入方式

- **GitHub 项目链接**：直接输入远程仓库地址
- **本地项目路径**：指定本地目录

### 输出结果

- 自动分析项目代码结构
- 提取核心调用的关系链条（call chain）
- 生成一张可视化的源码全景图

### 应用场景

| 场景 | 说明 |
|------|------|
| 学习开源项目 | 快速了解项目整体逻辑架构 |
| 代码审查 | 梳理模块间的调用关系 |
| 项目维护 | 掌握遗留代码的执行流程 |
| 教学演示 | 展示项目核心逻辑链 |

## 技术原理

### 工作流程

```
用户输入（GitHub链接/本地路径）
       ↓
AI 自动分析代码
       ↓
识别函数/模块定义
       ↓
追踪调用关系链
       ↓
生成可视化全景图
```

### 核心能力

- **代码解析**：理解项目结构，识别入口点
- **调用链追踪**：从入口函数出发，追踪所有调用的函数
- **可视化渲染**：将分析结果以图形方式呈现
