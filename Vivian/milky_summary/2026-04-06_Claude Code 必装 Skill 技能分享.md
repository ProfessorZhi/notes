# Claude Code 必装 Skill 技能分享

## 概述

本期视频分享三个能大幅提升 Claude Code 使用效率的 Skill 技能：

| Skill 名称 | 主要功能 | 推荐指数 |
|-----------|---------|---------|
| superpowers | 开发流程规范化（需求确认→技术选型→文档→代码） | ⭐⭐⭐⭐⭐ |
| claude-hud | 终端底部实时显示上下文额度、调用量等状态信息 | ⭐⭐⭐⭐⭐ |
| skill-creator | Claude 官方出品的自定义 Skill 创建工具 | ⭐⭐⭐⭐ |

---

## 1. superpowers - 开发流程规范化技能包

### 核心定位

> 它不是一个 Skill，而是一个 Skill 技能包，里面集成了一整套开发需要用到的 Skill。

### 四阶段开发流程

```
需求确认 → 技术选型 → 文档编写 → 代码开发
```

#### 第一阶段：需求确认（brainstorm）

- 当你描述大致需求后，自动调用 brainstorm 技能
- 连续向你提问几个具体问题
- 采用选择题形式确认需求细节

#### 第二阶段：技术选型

- 列出几套技术架构方案
- 给出采纳建议
- 如果没有特别偏好，直接选择推荐的方案即可

#### 第三阶段：文档编写

自动生成并保存两份文档到项目中：

| 文档类型 | 包含内容 |
|---------|---------|
| 设计文档 | 整体设计思路、架构图 |
| 技术实现方案 | 项目结构、核心代码规划 |

#### 第四阶段：代码开发

支持两种开发模式：

| 模式 | 特点 |
|-----|------|
| 普通模式 | 顺序执行任务 |
| subagent 模式 | 同时派出多个 agent 并行开发，速度更快 |

**开发过程中的保障机制：**

- 每完成一个任务都会进行 **code review**（代码审查）
- 全部完成后进行整体审查
- 自动提交到 git 仓库

### 安装方式

在 Claude Code 中运行命令创建新的 skill：

```bash
# 在 Claude Code 中执行
/install superpowers
```

## 2. claude-hud - 实时状态栏插件

### 功能作用

在终端底部添加状态栏，实时显示你最关心的 Claude Code 运行状态信息。

### 显示信息一览

| 显示项 | 说明 |
|-------|------|
| 上下文额度 | 当前对话的上下文剩余量 |
| 短期调用量 | 5 小时内剩余调用量 |
| 长期调用量 | 一周内剩余调用量 |
| 当前模型 | 正在使用的模型名称 |
| 订阅套餐 | 你的订阅计划类型 |

### 核心价值

- **防止额度耗尽**：避免"编译到 99% 突然没额度了"的尴尬情况
- **主动清理**：看到上下文快满时，可主动使用 `compact` 命令清理

### 注意事项

> ⚠️ 注意：使用第三方模型无法使用此功能，只能使用官方模型。

### 类似替代方案

> 弹幕提示：输入 `/statusline` 有类似的效果

## 3. skill-creator - 自定义 Skill 创建工具

### 工具背景

- Claude 官方出品的 Skill 创建工具
- 定位：用来创建 Skill 的 Skill

### 为什么需要自定义 Skill

Claude Code 本身的能力是**通用的**，但你的工作流程是**个人的**。

**示例场景：**
每次写公众号文章都要润色、排版、调整风格。与其每次都跟 Claude 解释一遍要求，不如直接做一个专属的 Skill。

### 使用流程

1. 告诉 Claude Code：使用 skill-creator 创建一个公众号文章排版的 Skill
2. 说明目标：把 Markdown 格式的文章排版成可以直接发布的公众号文章
3. skill-creator 会提问几个问题确认需求
4. 按照规范自动创建好排版 Skill

### 后续使用

创建完成后，只需说：

```
帮我把这篇 Markdown 文章排版
```

Claude Code 会自动调用这个含有固定排版样式的 Skill，无需重复说明排版规则。

## 总结

这三个 Skill 分别解决了不同层面的问题：

- **superpowers**：解决开发流程混乱的问题，从需求到代码全程规范化
- **claude-hud**：解决"闭眼开车"的问题，实时掌握资源消耗状态
- **skill-creator**：解决重复劳动问题，把个人工作流固化为可复用的工具
