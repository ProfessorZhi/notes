# 全网优质 Claude Code Skill 技能资源网站推荐

本篇笔记整理自 UP 主「我是阿众」的分享视频，介绍 5 个可以找到 Claude Code 以及其他 AI 编程工具（如 OpenClaw/Monk）适配的 Skill 技能资源的网站。

---

## 一、SkillsMP（skillsmp.com）

> 可能是全网最大的 Skill 技能分享网站

| 指标 | 数据 |
|------|------|
| 收录数量 | **73 万+** 免费 Skill |
| 搜索功能 | 支持关键词搜索 |
| 分类维度 | **23 个职业大类** |

**核心特点：**

- **关键词搜索**：可按关键词直接搜索，例如搜索"微信"能找到与微信或公众号文章写作相关的 Skill
- **职业分类导航**：将所有 Skill 按职业分为 23 个大类，包括但不限于：
  - 计算机类
  - 商业类
  - 艺术设计类
  - 各具体职业细分

**适用场景**：需要精准定位特定功能 Skill，或按职业需求批量浏览相关技能资源。

---

## 二、Skills.sh（skills.sh）

| 指标 | 数据 |
|------|------|
| 收录数量 | **9 万+** 免费 Skill |

**核心特点：**

- **Skill 排行榜**：首页直接展示热门 Skill 排名
- **安装量统计**：显示各 Skill 的累计安装次数
- **24 小时趋势**：实时追踪最近 24 小时内的安装趋势变化

**适用场景**：想了解当前社区最流行、最值得安装的 Skill，快速掌握近期热点趋势。

---

## 三、AgentSkill.sh（agentskill.sh）

| 指标 | 数据 |
|------|------|
| 收录数量 | **10 万+** Skill |

**核心特点：**

- **多维度分类**：按角色、平台、标签等多维度组织整理 Skill
- **质量评分系统**：部分 Skill 下方直接显示**质量评分**
- **安全评分系统**：提供**安全评分**，帮助判断 Skill 是否可信、是否存在风险

**适用场景**：在安装陌生 Skill 前需要评估其可信度，或想按特定角色/平台筛选匹配的工具。

---

## 四、Lobehub Skills（lobehub.com/skills/）

| 指标 | 数据 |
|------|------|
| 收录数量 | **25 万+** Skill |

**核心特点：**

- **方向筛选功能**：支持按具体应用方向筛选，包括但不限于：
  - 编程开发
  - 文档处理
  - 市场营销
- **首页内容模块**：
  - 安装最多
  - 高分推荐
  - 最近更新
  - 精选合集

**适用场景**：有明确使用方向，希望高效筛选出最合适、最热门 Skill 的场景。

---

## 五、SoFindAI（sofindai.com）

| 类型 | AI 工具搜索导航网站 |
|------|---------------------|

**定位说明：** 该网站本身不直接分享 Skill，而是一个 **AI 工具聚合导航平台**，收录了大量分享 Skill 资源的网站。

**核心价值：**

- 汇总多个 Skill 网站的入口
- 提供统一的浏览和搜索入口
- UP 主本人推荐：「前面跟大家分享的这 4 个网站都是从这里找的，非常好用」

**适用场景：** 想要一个入口统一查找和比较多个 Skill 网站资源。

## 网站对比汇总

| 网站 | 收录量 | 搜索 | 分类方式 | 特色功能 |
|------|--------|------|----------|----------|
| **SkillsMP** | 73 万+ | ✅ 关键词 | 职业分类（23类） | 职业导向浏览 |
| **Skills.sh** | 9 万+ | ✅ | 排行榜展示 | 安装量趋势、24h 热点 |
| **AgentSkill.sh** | 10 万+ | ✅ | 角色/平台/标签 | 质量评分、安全评分 |
| **Lobehub Skills** | 25 万+ | ✅ | 方向筛选 | 安装量/高分/精选合集 |
| **SoFindAI** | 导航聚合 | ✅ | 网站目录 | 汇总多个 Skill 站点 |

---

## 使用建议

1. **精准找 Skill** → 首选 SkillsMP（量大、关键词搜索精准）
2. **追热点趋势** → 首选 Skills.sh（看排行榜和 24h 趋势）
3. **安全筛选** → 首选 AgentSkill.sh（参考质量/安全评分）
4. **方向筛选** → 首选 Lobehub Skills（按开发/文档/营销方向）
5. **一站式导航** → SoFindAI（聚合多个站点入口）
