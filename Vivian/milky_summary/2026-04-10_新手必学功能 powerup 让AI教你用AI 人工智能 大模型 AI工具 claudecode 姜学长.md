# Claude Code PowerUp 交互式学习模式详解

## 什么是 PowerUp

PowerUp 是 Claude Code 推出的交互式学习模式，旨在让 AI 教你如何使用 AI 工具。它解决了传统学习方式的痛点：

- **传统方式**：需要切屏查看文档，或使用第三方软件查阅资料
- **PowerUp 方式**：直接在终端内完成学习，动画演示 + 手把手带练

## 使用方法

```bash
claude-code --powerup
```

输完命令按回车，即可启动交互式课程。

## 核心特点

| 特点 | 说明 |
|------|------|
| 边看边练 | 不是扔一堆文字让你背诵 |
| 逐步演示 | 像有人坐在旁边手把手教学 |
| 即时反馈 | 每完成一项显示绿色勾 ✅ |
| 游戏化 | 10个关卡逐个解锁，成就感递增 |

## 教学内容：10 个核心模块

### 模块 1：Talk to Your Code Base

**功能**：基础文件引用

**操作**：

- 使用 `@` 符号引用文件
- 支持单个文件引用
- 支持整个文件夹引用
- 引用后 AI 可直接读取上下文

### 模块 2：Shift with Most

**功能**：切换响应模式

**操作**：

- 按住 `Shift + Tab` 切换模式
- 快速模式：让 AI 快速响应
- 深度思考模式：处理复杂任务

### 模块 3：Undo Anything

**功能**：撤销操作

**操作**：

- 使用 `/reject` 指令
- 代码改错时一键回退
- 不想执行某个操作时快速撤销

### 其他 7 个模块

视频中未详细展开，属于进阶内容，涵盖：

- 斜杠命令的高级用法
- Bash 命令执行
- 后台任务运行
- 其他进阶操作

## 学习路径建议

1. 按顺序完成每个模块
2. 每完成一个模块按回车确认
3. 等待绿色勾 ✅ 出现
4. 全部完成后对该工具有了全面了解

## 适用人群

- 被 Claude Code "黑盒子"劝退的新手
- 想系统学习 CLI 工具的开发者
- 偏好"实战派"学习方式的用户（边看边练而非死记硬背）
