# 2026年 Python 入门零基础教程 - 知识框架

本教程通过 100 个核心分 P 视频，系统地涵盖了从 Python 环境搭建到高级实战的全过程。以下是根据 100 个分 P 标题整理的逻辑框架：

## 第一阶段：【基础篇】核心语法与环境 (P1 - P31)
*   **起步与环境**：
    *   Python 介绍、安装与配置 (Windows/Mac)。
    *   PyCharm 集成开发环境与 AI 插件的使用。
*   **基础语法**：
    *   注释、输出函数 (`print`)、输入函数 (`input`)。
    *   变量定义、赋值与命名规范。
    *   运算符：算术、比较、逻辑运算符详解。
*   **数据类型与容器**：
    *   字符串 (str)：运算、索引、切片、转义与格式化、常用内建函数。
    *   列表 (List)、元组 (Tuple)、字典 (Dict)、集合 (Set) 的特性及常见操作。
*   **流程控制**：
    *   条件分支：`if` 语句、多分支判断、分支嵌套。
    *   循环结构：`while` 循环、`for` 循环、循环嵌套、`break` 与 `continue` 控制。
*   **函数基础**：
    *   函数定义、调用、参数传递、返回值。
    *   变量作用域 (全局 vs 局部)。
    *   模块与包的概念及使用。
*   **错误处理**：
    *   异常处理 (`try...except`) 与抛出自定义异常。

## 第二阶段：【进阶篇】面向对象与高级特性 (P32 - P58)
*   **面向对象 (OOP)**：
    *   类与对象、封装、继承、多态。
    *   静态方法、类方法、单例模式、魔术方法 (`__init__` 等)。
*   **文件与目录操作**：
    *   文件读写、写入、文件路径处理。
    *   目录常用操作 (os/pathlib 相关内容)。
*   **高级语法特性**：
    *   正则表达式：核心规则、进阶规则、常用函数、贪婪与非贪婪匹配。
    *   闭包与装饰器 (Decorator)。
    *   迭代器 (Iterator) 与 生成器 (Generator)。
*   **多任务并发编程**：
    *   多线程 (`threading`)：基础、核心同步机制。
    *   多进程 (`multiprocessing`)：基础、进程间通信、进程池。
    *   协程 (`asyncio`)：基础、工具与总结。

## 第三阶段：【练习篇】100道实战练习题 (P59 - P68)
*   通过 10 组视频（每组 10 题）进行高强度巩固练习，覆盖从基础逻辑到算法实现的 100 个典型案例。

## 第四阶段：【实战篇】网络爬虫技术 (P69 - P87)
*   **基础理论**：爬虫原理、HTTP/HTTPS 协议。
*   **请求与解析**：
    *   `requests` 库基本使用、参数传递、模拟登录。
    *   数据提取：XPath 解析、JSONPath 解析。
*   **动态内容抓取**：
    *   `Selenium` 入门、进阶使用、反爬破解方案。
*   **进阶反爬破解**：
    *   JS 逆向破解实战 (滑动验证码轨迹、反调试破解)。
    *   验证码识别、IP 池与 Cookie 池构建。
    *   字体反爬破解。
*   **综合项目**：JS 逆向综合实战项目。

## 第五阶段：【分析篇】数据分析入门 (P88 - P100)
*   **环境与需求**：数据分析环境搭建、需求分析方法。
*   **NumPy**：数组创建、索引切片、矩阵运算、常用数学函数。
*   **Pandas**：
    *   DataFrame 常见操作、文件读取与保存。
    *   数据清洗与初步分析。
